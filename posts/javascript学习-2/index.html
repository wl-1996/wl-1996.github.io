标准写法：

}else if(){

}else if(){

}else{

switch case 语句

存在的意义：在某种情况下简化if...else...的写法

标准语法：

switch(待检测值){
   case 值1 :
      值1 与 待检测值 相同时做的事情；
      break;
   case 值2 :
      值2 与 待检测值 相同时做的事情；
      break;
   case 值3 :
      值3 与 待检测值 相同时做的事情；
      break;
   default :
      默认要做的事情；
      break;
}

三元运算符（即问号冒号表达式）

? : 是一组运算符，这是 JS 中唯一一个需要三个元素参加的运算符。

语法：

条件 ？ val1 : val2

表达式的值看条件是 true 还是 false 。如果条件是 true,那么表达式的值就是 val1 ,如果条件是 false,那么表达式的值就是 val2。
